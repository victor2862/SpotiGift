# SpotiGift

Pequeno site estático que imita a interface do Spotify móvel para ser usado como presente de aniversário. A ideia é colocar fotos do casal como capas de álbuns e transformar letras de músicas em mensagens personalizadas para minha namorada, aproveitando a estética e as features do app: playlists, telas de faixa em reprodução, cards de artistas, destaques e seções de descobertas. O site será hospedado no GitHub Pages e feito todo em JavaScript/HTML/CSS, ajustando apenas o tamanho em telas maiores (sem necessidade de responsividade avançada).
